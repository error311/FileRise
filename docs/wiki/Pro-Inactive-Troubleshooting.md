# Pro inactive troubleshooting

If Pro shows as inactive, use this quick checklist.

## 1) License saved

- Go to Admin -> FileRise Pro.
- Paste your license key (FRP1...) and click Save license.

## 2) Pro bundle installed

- Upload and install the Pro bundle ZIP.
- Confirm `users/pro/bootstrap_pro.php` exists and is readable.

## 3) Version mismatch

- Make sure your FileRise core version matches the Pro bundle version you installed.
- If unsure, update core first, then install the matching Pro bundle.

## 4) Updates window ended

- Your current Pro bundle keeps working.
- If you downloaded a newer bundle after your updates window, renew to get a new license.
