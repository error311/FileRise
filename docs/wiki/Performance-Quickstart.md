# Performance quickstart

Use this checklist for large trees or busy servers. See the full tuning guide for details.

## Checklist

- Use FR_IGNORE_REGEX to skip noisy or massive folders.
- Keep `metadata/` on fast local storage (avoid slow network disks).
- Limit public exposure and reduce unnecessary scans.
- Upgrade FileRise core and Pro to the latest versions you are eligible for.
